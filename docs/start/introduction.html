简介 ​

前置准备 ​

环境要求

开始之前建议将Bun更新到最新版本

前置知识

本框架基于Elysia编写，需要对它有一定的了解

bash
bun upgrade

启动项目 ​

bash
git clone xxx
bash
暂不支持

注意

注意存放代码的目录及所有父级目录最好不要存在中文、韩文、日文以及空格，否则可能会出现预料外的问题。

安装依赖 ​

bash
# 进入项目文件夹
cd mango

# 安装依赖
bun install

注意

  • 如果使用bun安装依赖失败，请更换镜像源

运行项目 ​

执行以下命令运行项目:

bash
bun run dev

此时，你会看到类似如下的输出代表项目运行成功：
